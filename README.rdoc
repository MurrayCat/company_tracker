== README

This is a basic rails app.

Getting started:

1) Terminal commands (Start here)
	bundle install
	#Install gems
	rake db:migrate
	#Creates dbs 
	bundle exe guard 	
	#runs guard for tests
	rails s
	#strats the server


2) Using the app
	Start the server:
		rails s
	Go to:
		http://0.0.0.0:3000
		Sign up or Login >> Create a company >>Add branches to that company
